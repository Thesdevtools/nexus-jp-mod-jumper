# nexus-jp-mod-jumper

**English Notice:**  
This browser extension is primarily intended for Japanese users of Skyrim SE and Fallout 4 mods.  
It adds a button on Nexus Mods pages to quickly jump to Japanese mod database sites (e.g., スカイリムMODデータベース).  
The full documentation and detailed guide are written in Japanese on Note.  
If you're not targeting Japanese mod communities, this might not be useful for you.

→ [詳細な使い方・解説はこちら（Note記事・日本語）](https://note.com/thesdevtools/n/n7873f91505f8)

---

## 簡単な概要（日本語）

Nexus Modsのページに「📖 Modデータベースで見る」ボタンを追加して、日本語のMODまとめサイト（主にスカイリムMODデータベースなど）にワンクリックで飛べるブラウザ拡張機能です。

- Skyrim Special Edition メイン対応  
- Fallout 4 にも一部対応  

MO2ユーザーや新作MODチェックが多い人に便利だと思います！

詳細な説明・スクショ・使用感などは、上記のNote記事をご覧ください。

## インストール手順（超簡単）

1. このリポジトリをZIPでダウンロード  
2. 解凍してフォルダを開く  
3. ブラウザで `chrome://extensions/` を開く（Edge/Firefoxも同様）  
4. 右上の「デベロッパーモード」をON  
5. 「パッケージ化されていない拡張機能を読み込む」で解凍したフォルダを選択  

これで即反映されます！  
（導入直後はNexusのページをリロードしてください）

動作や内容が気になる方は、`manifest.json` などのファイルを確認するか、ChatGPTなどにコードを貼ってチェックしてください。シンプルな処理しかしていません。
